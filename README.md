AnkoA
=====

FFMPEG Command Line Easy Encoding Tools

![AnkoA](http://i.imgur.com/BlG3BNs.png "AnkoA")


## Tools description :

* FFMPEG easy encoding
* Thumbnails Generator
* NFO Generator
* Genprez Upload
* Auto make .torrent
